Cafeteria

Descrição

  • Thomas

  • Ver Site

O projeto Cafeteria é um template simples desenvolvido pelo DevClub para uma página web de cafeteria. Ele utiliza HTML e CSS para criar uma interface básica que apresenta informações importantes sobre a cafeteria, como cardápio, avaliações de clientes, endereço e localização no mapa.

Tecnologia

  • HTML5
  • CSS3
